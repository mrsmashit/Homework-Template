# LaTeX Homework Template

This repository contains a custom LaTeX document class (`homework.cls`) designed for formatting homework assignments in a clean and professional manner.

## Features

- Professional homework formatting
- Support for problem and solution environments
- Automatic numbering of problems
- Built-in support for equations, figures, and tables
- Customizable header with course information

## Usage

1. Place `homework.cls` and `homework.tex` in the same directory.
2. Compile the `homework.tex` file to get `homework.pdf`
3. `assignment.tex` does not use this class, it is another template using the article class.

## Dependencies

- A modern LaTeX distribution (TeXLive, MikTeX, or MacTeX)
- Basic LaTeX packages (automatically loaded by the class)

## Compilation

Compile your document using:

- pdfLaTeX
- XeLaTeX
- LuaLaTeX

Example:

```bash
pdflatex your-homework.tex
```
